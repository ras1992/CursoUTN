Ejercicios para practicar JS

50 Ejercicios Sencillos de Lógica de Programación Básica en Javascript

  1. Crea una función que sume dos números y retorne el resultado.
  2. Crea una función que reste dos números y retorne el resultado.
  3. Crea una función que multiplique dos números y retorne el resultado.
  4. Crea una función que divida dos números y retorne el resultado.
  5. Crea una función que calcule el área de un círculo dado su radio.
  6. Crea una función que calcule el área de un triángulo dado su base y altura.
  7. Crea una función que calcule el área de un rectángulo dado su base y altura.
  8. Crea una función que calcule el área de un cuadrado dado su lado.
  9. Crea una función que calcule el perímetro de un círculo dado su radio.
  10. Crea una función que calcule el perímetro de un triángulo dado sus tres lados.
  11. Crea una función que calcule el perímetro de un rectángulo dado sus dos lados.
  12. Crea una función que calcule el perímetro de un cuadrado dado su lado.
  13. Crea una función que convierta grados Celsius a Fahrenheit.
  14. Crea una función que convierta grados Fahrenheit a Celsius.
  15. Crea una función que calcule el máximo de dos números.
  16. Crea una función que calcule el mínimo de dos números.
  17. Crea una función que calcule el máximo de tres números.
  18. Crea una función que calcule el mínimo de tres números.
  19. Crea una función que calcule si un número es par o impar.
  20. Crea una función que calcule si un número es positivo, negativo o cero.
  21. Crea una función que calcule la longitud de una cadena de texto.
  22. Crea una función que convierta una cadena de texto a mayúsculas.
  23. Crea una función que convierta una cadena de texto a minúsculas.
  24. Crea una función que calcule la suma de los primeros N números pares.
  25. Crea una función que calcule la suma de los primeros N números impares.
  26. Crea una función que calcule la raíz cuadrada de un número.
